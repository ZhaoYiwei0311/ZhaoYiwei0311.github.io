435. Non-overlapping Intervals

Intro

Given an array of intervals intervals where intervals[i] = [starti, endi], return the minimum number of intervals you need to remove to make the rest of the intervals non-overlapping.

Solution

1.Greedy

This question is quite similar to 452. Minimum Number of Arrows to Burst Balloons. The new point is to understand that to count remove is just to count overlapping intervals and update boarders accordingly like the picture below.

And of course, we need to sort array by its left or right boarder first. I choose left boarder this time.

763. Partition Labels

Intro

You are given a string s. We want to partition the string into as many parts as possible so that each letter appears in at most one part.

Note that the partition is done so that after concatenating all the parts in order, the resultant string should be s.

Return a list of integers representing the size of these parts.

Solution

1.Greedy

It is a tricky question.

We must first record the furthest distance of each letter in the string and then do partition until the current iteration index equals to the furthest distance of letter iterated.

56. Merge Intervals

Intro

Given an array of intervals where intervals[i] = [starti, endi], merge all overlapping intervals, and return an array of the non-overlapping intervals that cover all the intervals in the input.

Solution

1.Greedy
