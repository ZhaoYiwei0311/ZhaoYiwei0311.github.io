1049. Last Stone Weight II

Intro

You are given an array of integers stones where stones[i] is the weight of the ith stone.

We are playing a game with the stones. On each turn, we choose any two stones and smash them together. Suppose the stones have weights x and y with x <= y. The result of this smash is:

If x == y, both stones are destroyed, and
If x != y, the stone of weight x is destroyed, and the stone of weight y has new weight y - x.
At the end of the game, there is at most one stone left.

Return the smallest possible weight of the left stone. If there are no stones left, return 0.

Solution

1.DP

The point is to divide stones into 2 parts, the difference of which gets closest.

The value and weight are both stones[i]. The size of pack can be set to the sum of stones divided by 2.

The last element of dp array is the first stone pack with heaviest stones we can pick and the sum minus this element means another pack that is left.

494. Target Sum

Intro

You are given an integer array nums and an integer target.

You want to build an expression out of nums by adding one of the symbols ‘+’ and ‘-‘ before each integer in nums and then concatenate all the integers.

For example, if nums = [2, 1], you can add a ‘+’ before 2 and a ‘-‘ before 1 and concatenate them to build the expression “+2-1”.
Return the number of different expressions that you can build, which evaluates to target.

Solution

1.DP

It seems there is no concern about dp but actually by deduction we know there is a connection.

The point is still to divide array into 2 parts. One is left with all positive numbers, the other one is right whose numbers are supposed to be negative. And we still have the sum that is left + right. The target given can be expressed by left - right, which can be converted to left - (sum - left) = target and finally, left = (target + sum) / 2.

Exception occurs when (target + sum) / 2 is not even or sum is smaller than target, and in these cases we will directly return 0.

Then we must understand that this is a question asking for combination and a common formula for combination is dp[j] += dp[j - nums[i]]. But to make this formula work, we must initialize dp[0] to 1, otherwise sum will always remain 0.

474. Ones and Zeroes

Intro

You are given an array of binary strings strs and two integers m and n.

Return the size of the largest subset of strs such that there are at most m 0’s and n 1’s in the subset.

A set x is a subset of a set y if all elements of x are also elements of y.

Solution

1.DP

This is also a 0-1 Knapsack Problem because all elements can only be picked once. And we have two dimensions to simultaneously take into account. One is the number of 1 and the other is the number of 0.
